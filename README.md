# Federated Learning for Energy-Efficient Cloud Resource Allocation

Privacy-preserving machine learning system for optimizing cloud resource allocation and energy consumption across distributed data centers.

## Features

- Federated learning across multiple data center nodes
- Energy consumption modeling and optimization
- Privacy-preserving gradient aggregation
- Dynamic resource scaling recommendations
- Benchmark comparison with centralized approaches

## Tech Stack

Python, TensorFlow Federated, PySpark, NumPy, Matplotlib

## Getting Started

### Prerequisites

- Python 3.8+
- pip

### Installation

```bash
git clone https://github.com/karthik-idikuda/Federated-Learning-Based-Energy-Efficient-Cloud-Resource-Allocation.git
cd Federated-Learning-Based-Energy-Efficient-Cloud-Resource-Allocation
pip install -r requirements.txt
```

### Usage

```bash
python train_federated.py
```
